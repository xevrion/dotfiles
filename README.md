# xevrion's dotfiles

My personal dotfiles for Fedora 43 + Hyprland + Noctalia setup.

## Stack
- OS: Fedora 43
- WM: Hyprland
- Shell: Noctalia
- Terminal: Ghostty
- Editor: Neovim (LazyVim)
- Shell: Zsh + Powerlevel10k
- Launcher: Rofi

## Install
\`\`\`bash
git clone https://github.com/xevrion/dotfiles ~/dotfiles
cd ~/dotfiles
stow hyprland ghostty nvim zsh rofi noctalia
\`\`\`
